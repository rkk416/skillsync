# SkillSync AI Development Rules

Every AI prompt must contain the following project context:

```text
Project Name: SkillSync
Language: Java
GUI: JavaFX
Database: PostgreSQL
Architecture: MVC
```

## Rules

1. Use the existing package structure.
2. Use the existing model classes.
3. Use the existing service interfaces.
4. Use the existing `DatabaseConnection`.
5. Use the existing PostgreSQL schema.
6. Generate code only for the assigned module.
7. Do not create duplicate models.
8. Do not modify the architecture.
